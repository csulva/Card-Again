# Card-Again

## Installation

It is  recommended to install requirements in a virtual environment (venv).
```bash
python3 -m venv venv
. venv/bin/activate
```
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Ragtime requirements.

For Production:
```bash
pip install -r requirements/heroku.txt
```
